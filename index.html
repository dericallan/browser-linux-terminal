Browser Linux Terminal

A real Alpine Linux, booted with v86 (x86 emulated via WebAssembly) and running entirely inside this tab. Nothing is sent to a server — standard Alpine utilities and the apk package manager are available, but there's no network access. Your work is saved automatically in this browser and restored next time you open this terminal; the Restart button intentionally wipes it and starts fresh.
